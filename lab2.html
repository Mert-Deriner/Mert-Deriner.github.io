Web Programlama — Lab 2

HTML ve CSS temellerini kullanarak hazırlanmış örnek sayfa

İçerik

Bu bölümde web programlamanın temel taşları olan HTML ve CSS hakkında bilgiler yer almaktadır. HTML, web sayfalarının iskeletini oluşturan bir işaretleme dilidir ve her modern web sitesinin temelinde bulunur.

CSS ise bu yapıya görsel kimlik kazandıran stil dilidir. Renkler, yazı tipleri, kenar boşlukları ve sayfa düzeni gibi pek çok görsel özellik CSS aracılığıyla kontrol edilir. İkisinin birlikte kullanımı profesyonel web sayfaları üretmeyi mümkün kılar.

Yan Menü / Bilgi

Bu alanda ek bilgiler ve faydalı bağlantılar paylaşılmaktadır. Dersin içeriğine yönelik kısa notlar burada yer alabilir.

HTML5 ve CSS3 günümüzde web standartlarının en güncel sürümleridir ve tüm modern tarayıcılar tarafından desteklenmektedir.
